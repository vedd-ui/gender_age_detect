# Gender Detection using CNN

## Overview
This project implements a Convolutional Neural Network (CNN) to classify gender from facial images using the UTKFace dataset.

## Dataset
UTKFace dataset (not uploaded due to size)
Download: https://www.kaggle.com/datasets/jangedoo/utkface-new

## Workflow
* Loaded and processed image dataset using OpenCV
* Extracted age and gender labels from filenames
* Resized images to 64x64
* Built CNN model using TensorFlow/Keras
* Trained model and evaluated performance

## Results
* Achieved ~77% accuracy on gender classification
* Model successfully predicts gender from unseen images

## Key Learnings
* Image preprocessing
* CNN architecture basics
* Handling memory constraints
* Model training and evaluation

## Tech Stack
Python, TensorFlow, Keras, OpenCV, NumPy, Matplotlib
